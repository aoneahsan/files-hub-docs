# FilesHub pricing

FilesHub is **free** to use. It is the zero-cost file-storage + utility API backend used across
the Aoneahsan/Zaions projects.

| Plan | Price | Includes |
|------|-------|----------|
| Free | $0 / forever | File upload/download/list/delete, per-file visibility, auto-expiry, scoped + restrictable API keys, transactional email (send/templates/schedules), and 50+ utility endpoints. |

- No usage-based billing. No paid tiers.
- Self-host your own FilesHub instance, or use the hosted one at https://fileshub.zaions.com.
- Default upload cap 10 MB (configurable per deployment).

Support continued development (optional): https://aoneahsan.com/payment
